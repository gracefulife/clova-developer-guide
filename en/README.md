# README
This documentation is a developer guide and API reference for the CIC and CEK platforms. The target audiences of this document are client developers using CIC to develop electronic devices and apps that link with Clova services and extension developers using CEK to provide online content and services.

<div class="note">
  <p><strong>Note!</strong></p>
  <p>Clova is under continuous development.  Thus, the information in this documentation is subject to change at any time.</p>
</div>

## Contacts
Contact the Clova partnership team for any inquiries about this documentation.
